# leukemiaDB
##Repository for the Acute Leukemia Database

This leukemia database is intended to provide a basis for researchers working in
the classification of acute leukemia. The database was created in collaboration
with physicians (from different hospitals) in order to implement a system for
the automatic morphological identification of acute leukemias from bone marrow 
cells images. The files included in this repository are:
- **L1L2M2M3M5.arff**, containing a weka file with all the attributes extracted from each segmented cell image
- **LeukemiaSegmentedImages.zip**, containing a set of folders with the segmented leukemia images used to extract the features of the L1L2M2M3M5.arff file.

For a description of how the data has been used and to **cite out work**, please refer to the following publication:
- PLOS ONE
